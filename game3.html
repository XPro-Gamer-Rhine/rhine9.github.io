Dark Soul 4 (Tribute)

Dark Soul or Soul Series is a popular game series. It's one of my favourite game as it is difficult and the way the game is built so the bugs are seen as features of the game rather than some annoyance. I love how difficul the bosses are and how the environment is interconnected. Not many games can create level that trace back to the starting position via different shortcuts. So as a game developer this project is me trying to make something similar to Dark Soul Game series.
I know i am far off from the real thing but it's alright. I will get there eventually.
